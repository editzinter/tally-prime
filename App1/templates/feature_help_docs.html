First-Time Setup

Welcome to TallyPrime. This guide will help you get started.

Step 1: Create a Company
Navigate to Company > Create to set up your company profile including name, address, and financial year dates.

Step 2: Configure Chart of Accounts
Set up your ledger groups and create necessary ledger accounts under appropriate groups (Assets, Liabilities, Income, Expenses).

Step 3: Enter Opening Balances
For existing businesses, enter opening balances for all ledger accounts to carry forward data from your previous financial year.

Step 4: Start Voucher Entry
Begin recording daily transactions using vouchers — Sales, Purchase, Receipt, Payment, Journal, and Contra.

Keyboard Shortcuts:

  • Alt+F1 — Detailed/Condensed view
  • Alt+F2 — Change period
  • Alt+F3 — Company menu
  • F11 — Features menu
  • F12 — Configuration menu
